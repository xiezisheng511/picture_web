# PicEdit Web — 产品需求文档 (PRD)

## 1. 产品概述

**产品名称**：PicEdit Web（图片编辑工具站）
**一句话定位**：免费、免登录、纯浏览器端的图片处理工具箱，专注"换底色 / 去水印 / 抠图"等高频场景。
**核心价值**：用户上传图片 → 几秒内得到处理结果 → 直接下载，无需注册、无需上传服务器（隐私友好）。
**商业模式**：Google AdSense 广告变现（展示广告 + 工具结果页推荐位）。

## 2. 目标用户

- **个人用户**：需要换证件照底色、清理自拍水印、做社交媒体配图。
- **电商卖家**：批量处理商品图，去除平台水印。
- **学生 / 求职者**：证件照、考试报名照、简历照。
- **海外华人**：双语支持，便于不同地区用户使用。

## 3. 核心功能（MVP）

| # | 功能 | 描述 | 实现方式 |
|---|------|------|----------|
| F1 | 换背景颜色 | 上传人像/物品图 → 选择纯色（红/蓝/白/自定义）→ 自动抠图并换底 | Canvas + 颜色阈值 / AI API |
| F2 | 去水印 | 选择水印区域 → 算法自动修复 | Canvas 修复（区域复制/均值） |
| F3 | 基础编辑 | 裁剪、缩放、旋转、压缩、格式转换 | Canvas |
| F4 | AI 一键抠图 | 高质量人像/物品抠图 | remove.bg / Replicate API |
| F5 | 双语切换 | 中英 UI 文案切换 | i18next |
| F6 | AdSense 广告位 | 顶部 banner、侧边栏、结果下方 | 广告脚本嵌入 |

## 4. 用户流程

```
进入首页
  ↓
[选择工具卡片] 或 [直接拖拽上传]
  ↓
工具页：上传图片
  ↓
调整参数（颜色选色器 / 区域选择 / 输出格式）
  ↓
点击「处理」
  ↓
预览结果（左右对比）
  ↓
下载（PNG / JPG / WebP）
  ↓
[广告位触发曝光]
```

## 5. 信息架构（页面结构）

```
/                  首页（Hero + 工具卡片网格 + SEO 内容）
/tools/bg-color    换背景色工具
/tools/remove-watermark  去水印工具
/tools/edit        基础编辑工具
/tools/ai-cutout   AI 一键抠图
/about             关于我们
/privacy           隐私政策（AdSense 审核必备）
/terms             服务条款
```

## 6. AdSense 广告位规划

| 位置 | 页面 | 类型 | 说明 |
|------|------|------|------|
| Top Banner | 全站 | Display Ads | 顶部 728x90 / 响应式 |
| Sidebar | 工具页 | In-Article / Display | 右侧 300x250 |
| Result-Below | 工具结果下方 | In-Article | 转化后展示 |
| Footer | 全站 | Display | 728x90 |

## 7. SEO 与内容策略

- 每个工具一个独立页面，含 H1、关键词、简短描述（300+ 字介绍）。
- 输出 `/sitemap.xml` 和 `/robots.txt`。
- 图片懒加载，meta description 完整。
- 隐私政策、服务条款页面（AdSense 必填）。

## 8. 成功指标（上线 3 个月目标）

- 日 UV：500+
- 平均处理时长 < 10 秒
- AdSense 通过审核并稳定展示
- 跳出率 < 60%

## 9. 风险与约束

- **AdSense 审核**：必须有原创内容、隐私政策、联系方式；至少上线 2-4 周稳定运行。
- **AI API 成本**：默认使用前端纯算法，AI 接口作为可选付费增强（避免免费滥用）。
- **隐私承诺**：强调"图片不上传服务器"，这是产品差异化卖点。
